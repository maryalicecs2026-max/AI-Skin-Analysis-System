# Advanced Beauty Care – AI-Powered Skincare Analysis System

## Overview
Advanced Beauty Care is an AI-powered skincare analysis platform that helps users understand their skin through image analysis and personalized recommendations.

## Features
- Skin Type Detection
- Skin Concern Analysis
- Personalized Product Recommendations
- Ingredient Knowledge Base
- AI-Powered Image Analysis
- Progress Tracking Dashboard
- QR-Based Mobile Scan
- PDF & JSON Reports

## Technologies Used
- React
- Flask
- Python
- YOLOv8
- OpenCV
- MediaPipe
- Firebase
- Cloudflare Pages

## Project Highlights
- Detects skin type and skin concerns
- Recommends suitable skincare products
- Explains skincare ingredients
- Tracks user skincare progress
- Supports mobile photo scanning
